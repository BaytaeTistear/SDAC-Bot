# SDAC Bot Minimal Release

Date: 2026-06-17

This is a one-file minimal release marker for the SDAC Bot repository.

Included:

- Release notes only

Not included:

- Discord token or `.env`
- SQLite database files such as `sdac.db`
- `media/` uploads
- `backups/`
- `venv/` or Python cache files

The full Ubuntu-ready upload bundle remains in the local `server/` folder until the project files are intentionally pushed.
