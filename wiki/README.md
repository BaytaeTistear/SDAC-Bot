# SDAC Project Wiki

This wiki is the project knowledge base for SDAC. Keep it updated when bot, server, dashboard, or app behavior changes.

## Categories

- [Bot](bot.md) - Discord commands, permissions, game/activity features, update announcements, and bot-owner controls.
- [Server](server.md) - Deployment, environment variables, backups, health checks, releases, and maintenance.
- [Dashboard](dashboard.md) - Web routes, Discord OAuth, roles, server selection, theme/layout editing, and admin tools.
- [App](app.md) - SDACCompanion, Android builds, signing, store install notes, and the future SDK app direction.

## Update Rules

- Mark update scope clearly as `Bot update`, `App update`, or `Bot and App update`.
- Push completed updates to GitHub.
- When a feature changes user-facing behavior, update the matching wiki category in the same change.
- Future app updates should remove the web-wrapper app path and move to the SDK app approach.
