# Bot

The SDAC bot is the Discord-facing side of the project. It handles media submissions, moderation commands, games, release notifications, and bot-owner administration.

## Core Areas

- Media submission slash commands and repost handling.
- Category and submit-channel configuration.
- Moderator/admin/server-owner/bot-owner command access.
- Anime-related games and activities, currently marked as experimental and removable in future updates.
- Official update announcements for new official releases.
- Discord-side reset and maintenance commands.

## Access Notes

- Bot Owner is the highest global role.
- Server Owner applies only within that server.
- Users can have different roles on different servers.
- Admins should only manage servers they belong to or have been granted access to.
