Short description (75 characters):
Discord media submissions, guessing games, and moderation in one dashboard.

Full description:
Sana-Chan is a companion app for Discord communities that use the Sana-Chan bot and dashboard at https://thelab.us.to.

Use Sana-Chan to open your community dashboard, review submitted media, vote on posts, manage guessing-game library items, and keep moderation workflows organized from your phone or tablet.

For community members, Sana-Chan makes it easier to browse submissions, vote, check guessing-game activity, view stats, and access account tools without digging through Discord commands.

For moderators and server owners, Sana-Chan supports review queues, removal reasons, quarantine workflows, server-scoped access, setup checks, maintenance views, invite guidance, release notices, and diagnostics.

For bot owners, Sana-Chan provides a mobile-friendly path into global operations such as release status, health checks, backups, job status, and server management.

Key features:
- Discord login through the hosted dashboard
- Server-scoped dashboard access
- Media submission browsing and voting
- Moderator review and quarantine tools
- Guessing-game library management
- Bulk guess import support through CSV and media ZIP files
- Setup and release checklist views
- App diagnostics and update notices
- Guided bot invite flow

Sana-Chan is designed for communities that want a cleaner way to run Discord media events, screenshot sharing, anime activities, guessing games, and moderation without requiring every action to happen through slash commands.

A Discord account and access to a server using the Sana-Chan bot may be required for most features.
